Métodos de abogado - Perfil

Mi perfil#

Editar perfil#

PUT /api/user/me

Authorization: Customer

Content-Type: aplication/json

Request

{
"name": "Jorge",
"lastName": "Negrete",
"email": "jorgenegrete@gmail.com",
}

Response: 204 NoContent

AtributoValidaciones
namestring, min:3, max:150, required
lastNamestring, min:3, max:150, required
emailstring, email, required
imageobject,optional,
ValidacionesCódigo de respuestaCódigo de mensaje
Si el correo ya está registrado como usuario (cualquier modalidad)409email_in_use

Flujo Exitoso#

  1. Se actualiza la información del usuario en base.
  2. Se actualiza la información del usuario en firebase.
  3. Si se actualizo el correo, se envía un correo al usuario con el cual podrá establecer su contraseña.
